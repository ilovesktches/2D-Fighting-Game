# 2D Fighting Game

This is a pixel-style 2D fighting game built with Python and Pygame.  
You control a fighter who can punch and kick, facing off against a CPU opponent.

## Features
- Player movement and controls  
- Punch and kick attacks  
- CPU opponent with basic AI  
- Enemies that reset when hit or fall  
- Gradually increasing difficulty

## How to Play
1. Run the game using Python:  
   ```bash
   python3 main.py
